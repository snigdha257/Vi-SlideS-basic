# 🎓 Vi-SlideS Documentation

## Overview
Vi-SlideS is a real-time classroom interaction platform where students can ask questions and teachers can respond manually or using AI.

---

## Features

- Google Authentication (Teacher / Student)
- Session creation with unique code
- Real-time question submission
- AI-assisted answers
- Class mood tracking
- Session summary

---

## Workflow

1. Teacher creates session  
2. Students join using code  
3. Students submit questions  
4. Teacher answers or uses AI  
5. Session ends → summary generated  

---

## Tech Stack

- Frontend: React  
- Backend: Node.js + Express  
- Database: MongoDB  
- Realtime: Socket.IO  

---

## Unique Features

- QR-based session join  
- Teacher-controlled AI  
- Real-time interaction  

---

## Conclusion

Vi-SlideS enhances classroom engagement using real-time communication and AI.
